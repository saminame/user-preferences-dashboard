# User Preferences Dashboard

A TypeScript front-end project demonstrating state-driven UI updates, validation, debounced autosave, and persistence.

## Features
- Profile inputs with email validation
- Theme toggle (light/dark) with instant UI updates
- Debounced autosave to localStorage
- Reset to defaults
- Accessible save status messaging (aria-live)

## Tech
TypeScript, HTML, CSS, Vite

## Run locally
npm install
npm run dev
